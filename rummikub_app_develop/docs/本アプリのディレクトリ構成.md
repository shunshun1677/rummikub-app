# 本アプリのディレクトリ構成

このドキュメントでは、ラミーキューブアプリの現在の `src` 配下の構成と、各ディレクトリの責務をまとめます。

## 現在の構成

```text
src
├── assets
│   ├── icon
│   │   ├── react.svg
│   │   └── vite.svg
│   └── img
│       └── hero.png
├── features
│   └── rummikub
│       ├── _shared
│       │   └── types
│       │       └── types.ts
│       └── game
│           ├── components
│           │   ├── Controls.tsx
│           │   ├── CpuHand.tsx
│           │   ├── GameBoard.tsx
│           │   ├── PlayerHand.tsx
│           │   ├── StatusBar.tsx
│           │   ├── TileSetView.tsx
│           │   └── TileView.tsx
│           ├── logics
│           │   ├── cpu.ts
│           │   └── gameLogic.ts
│           ├── styles
│           │   └── game.css
│           └── index.tsx
├── pages
│   └── rummikub
│       └── index.tsx
├── App.tsx
├── index.css
└── main.tsx
```

## トップレベル

| パス | 責務 |
| --- | --- |
| `src/main.tsx` | React アプリの起動処理を持つエントリーポイント |
| `src/App.tsx` | アプリ全体の入口。現在は `RummikubPage` を表示する薄いコンポーネント |
| `src/index.css` | アプリ全体にかかるベース CSS |
| `src/assets` | 画像やアイコンなどの静的アセット |
| `src/pages` | 画面単位の入口 |
| `src/features` | 機能単位の実装 |

## `assets`

`assets` には、アプリで利用する静的ファイルを配置します。

| パス | 責務 |
| --- | --- |
| `src/assets/icon` | SVG アイコンなどのアイコン系アセット |
| `src/assets/img` | 画像ファイル |

## `pages`

`pages` には、画面として表示する入口コンポーネントを配置します。

| パス | 責務 |
| --- | --- |
| `src/pages/rummikub/index.tsx` | ラミーキューブ画面のページ入口。`features/rummikub/game` を呼び出す |

ページ層には、画面単位の繋ぎ込みだけを置きます。ゲームの状態管理、描画部品、ルール処理は `features` 配下に置きます。

## `features/rummikub`

`features/rummikub` は、ラミーキューブ機能全体の実装を管理します。

| パス | 責務 |
| --- | --- |
| `src/features/rummikub/_shared` | ラミーキューブ機能内で共有する型や共通処理 |
| `src/features/rummikub/game` | 実際のゲーム画面とゲーム進行に関する実装 |

## `features/rummikub/_shared`

複数のサブ機能で共有されるファイルを配置します。

| パス | 責務 |
| --- | --- |
| `src/features/rummikub/_shared/types/types.ts` | タイル、セット、ゲーム状態、CPU 手番などの型定義 |

今後、ゲーム履歴や設定画面などが増えて同じ型を使う場合も、共有型はここに置きます。

## `features/rummikub/game`

現在のゲーム本体です。

| パス | 責務 |
| --- | --- |
| `src/features/rummikub/game/index.tsx` | ゲーム画面のコンテナ。状態管理、ターン進行、各コンポーネントへの props 受け渡しを担当 |
| `src/features/rummikub/game/components` | ゲーム画面内で使う表示コンポーネント |
| `src/features/rummikub/game/logics` | React に依存しないゲームロジック |
| `src/features/rummikub/game/styles` | ゲーム画面専用 CSS |

## `features/rummikub/game/components`

ゲーム画面内で利用する UI コンポーネントを配置します。

| ファイル | 責務 |
| --- | --- |
| `Controls.tsx` | ターン終了、1枚引いて終了、新規セットなどの操作ボタン |
| `CpuHand.tsx` | CPU の手牌表示 |
| `GameBoard.tsx` | 場全体の表示とドロップ領域 |
| `PlayerHand.tsx` | プレイヤー手牌の表示 |
| `StatusBar.tsx` | ターン、山札、初回場出し、メッセージなどの状態表示 |
| `TileSetView.tsx` | 場にある1セット分の表示 |
| `TileView.tsx` | 牌1枚分の表示 |

## `features/rummikub/game/logics`

React コンポーネントに依存しない処理を配置します。

| ファイル | 責務 |
| --- | --- |
| `gameLogic.ts` | タイル生成、シャッフル、セット判定、得点計算、ターン検証など |
| `cpu.ts` | CPU の手番選択、候補セット生成、出せる牌の選択 |

UI に関係しないルール処理は、できるだけここに寄せます。

## 今後ファイルを追加する時の目安

| 追加したいもの | 配置先 |
| --- | --- |
| ゲーム画面内の表示部品 | `src/features/rummikub/game/components` |
| run / group 判定などのルール処理 | `src/features/rummikub/game/logics` |
| ラミーキューブ機能全体で使う型 | `src/features/rummikub/_shared/types` |
| 設定画面や履歴画面など新しい画面入口 | `src/pages/rummikub` または新しい `pages` 配下 |
| 設定機能、履歴機能などゲーム本体から分けたい機能 | `src/features/rummikub/settings` や `src/features/rummikub/history` |
| 共通画像、アイコン | `src/assets/img` または `src/assets/icon` |
